SpeeDOM

SpeeDOM Version 1.0.0
'Credential Store' is a Chrome extension used to store domain specific credentials and login by cliking on the desired credential.
Features:
  • One click redirection to domain.
  • Material UI.
  • Credentials sorting according to domain.
  • Environment specific badges.
  • Delete specific credential.
  • Reset entire data.
  • Automated login using saved credentials.
  • Toggle between doamin specific and all avaliable credentials.
