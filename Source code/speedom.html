SpeeDOM

SpeeDOM Version 1.0.0
'SpeeDOM' is a Web extension Calcutates the speed/time of DOM ready state.
Features:
  • Tab respective time logs.
  • Material UI.
  • DOM load time with timestamp.
  • Average spped calculation.
  • Auto log delete for tab close.
  • Save data to file.
